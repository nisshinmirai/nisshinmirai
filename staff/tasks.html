🔧 初回セットアップが必要です

このページを使うには、Firebase（無料）の設定が必要です。以下の手順で進めてください。

  1. Firebase コンソールを開く
  2. 「プロジェクトを追加」→ 任意の名前（例: nisshin-tasks）→ Googleアナリティクスは無効でOK
  3. 左メニュー「Firestore Database」→「データベースの作成」→ 本番環境モードを選択 → リージョンは asia-northeast1
  4. 「ルール」タブで以下に置き換えて公開: rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /tasks/{doc} { allow read, write: if true; } } }
  5. プロジェクト概要 → ウェブアプリ追加 (</>アイコン) → ニックネーム入力 → 表示されるfirebaseConfigをコピー
  6. staff/tasks.html 末尾の FIREBASE_CONFIG オブジェクトに貼り付け
  7. このページを再読み込み

設定後、URLを知っているメンバーは誰でも閲覧・編集できます。URLは安全に共有してください。
